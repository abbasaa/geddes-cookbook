## Quick Start

### Get Repo and Run Locally

```shell
git clone https://github.com/abbasaa/geddes-cookbook.git
cd geddes-cookbook/
npm install
gatsby develop
```

### Add a Recipe
```shell
cd content/blog/
mkdir newRecipe
cp sesame-steak-stirfry/* newRecipe/
cd newRecipe
vim index.md
```

### Updating the GHPages Build
```shell
npm run deploy
```
